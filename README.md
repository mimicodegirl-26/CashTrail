## ✨ Features

| Feature | Description |
|---------|-------------|
| 📝 **Quick Transaction Logging** | Add income or expenses in seconds with a clean form. |
| 📊 **Interactive Charts** | Visualize your spending by category (e.g., Food, Rent, Entertainment). |
| 🔍 **Search & Filter** | Find past transactions by title, amount range, or date. |
| 💾 **Local Storage** | All data stays on your device – no account needed, no cloud sync. |
| 🎨 **Responsive Design** | Works seamlessly on mobile, tablet, and desktop. |
| ⚡ **Blazing Fast** | Built with Vite + React for near-instant interactions. |
| 🔒 **Type Safety** | Full TypeScript coverage reduces bugs. |
| 🧪 **Modular Code** | Easy to extend with custom categories or export features. |

---

## 🎯 Why CashTrail?

- **No signup required** – open the app and start tracking immediately.
- **Privacy first** – your financial data never leaves your browser.
- **Minimalistic & intuitive** – no clutter, just the insights you need.
- **Open source** – audit the code, contribute, or customize it for your own use.

## 🚀 Getting Started

### Prerequisites
- **Node.js** v18 or higher ([Download](https://nodejs.org/))
- **npm** (comes with Node) or **yarn**

### Installation

```bash
# Clone the repository
git clone https://github.com/mimicodegirl-26/CashTrail.git
cd CashTrail

# Install dependencies
npm install
# or
yarn install
